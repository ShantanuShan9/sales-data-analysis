# sales-data-analysis
Exploratory Data Analysis of Superstore Sales Dataset using Python, Pandas and Seaborn.
Project Overview:-
This project performs Exploratory Data Analysis (EDA) on the Superstore Sales dataset to uncover business insights related to regional performance, category sales, and customer purchasing behavior.

🎯 Objective:-
To analyze sales performance across different regions, categories, and customer segments and identify key business insights.

🛠️ Tools & Technologies:-
Python
Pandas
NumPy
Matplotlib
Seaborn
Jupyter Notebook

Key Analysis Performed:-
Data cleaning and missing value handling
Sales aggregation by Region
Sales aggregation by Category
Average order value analysis
Order volume analysis
Data visualization

🔍 Key Insights:-
West region generates the highest total revenue due to high order volume.
South region has the highest average sales per order.
Technology category drives the maximum revenue.
Central region underperforms compared to other regions.

Future Scope:-
This dataset can be extended into a Machine Learning regression problem by predicting sales based on customer and product features.
